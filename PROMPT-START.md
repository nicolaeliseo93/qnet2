# Prompt Start

Usa questo prompt quando uno strumento non legge automaticamente le istruzioni del repository.

```text
Lavora all'interno di questo repository usando il sistema multi-agente locale.

Inizia leggendo:
- AGENTS.md
- standards/ai-rules.md
- standards/orchestration.md
- standards/routing-matrix.md
- standards/handoff-protocol.md

Poi classifica la richiesta, seleziona solo i file degli agenti rilevanti, segui le regole di orchestrazione e usa handoff strutturati tra le fasi.

Non inventare requisiti, API, campi database o comportamenti non documentati nel repository.
```
