# Codex Bootstrap

Se sei Codex e lavori in questo repository, parti da `AGENTS.md`.

Comportamento richiesto:

1. Leggi `AGENTS.md`
2. Leggi gli standard obbligatori elencati lì
3. Classifica la richiesta con `standards/routing-matrix.md`
4. Leggi solo i file degli agenti necessari al task
5. Segui `standards/orchestration.md`
6. Produci handoff strutturati usando `standards/handoff-protocol.md`

Non ignorare le istruzioni locali del repository a favore di default generici.
